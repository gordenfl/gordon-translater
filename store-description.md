# Gordon Translator - Chrome Extension Store Description

## 基本信息

**名称**: Gordon Translator  
**版本**: 1.2.0  
**类别**: 效率工具  
**语言**: 中文、英文  

## 详细描述

Gordon Translator 是一个现代化的 Google 翻译客户端，专为 Chrome 浏览器设计。它提供了简洁优雅的界面和强大的翻译功能，让您在任何网页上都能快速翻译文本。

### 主要功能

🌓 **智能主题切换**
- 支持浅色、深色和自动主题
- 根据系统时间自动切换主题
- 优雅的视觉体验

⚡ **快速翻译**
- 集成 Google Translate API
- 支持多种语言互译
- 实时翻译结果

🎯 **选择即翻译**
- 在任意网页选择文本即可翻译
- 自动打开侧边栏显示翻译结果
- 无需复制粘贴，提升效率

🎨 **现代化界面**
- 简洁美观的设计
- 响应式布局
- 流畅的用户体验

### 使用方法

1. 安装扩展后，点击工具栏图标打开翻译面板
2. 在任意网页选择文本，自动显示翻译结果
3. 支持手动输入文本进行翻译
4. 可切换不同主题模式

### 隐私保护

- 仅调用 Google Translate API 进行翻译
- 不收集或存储用户数据
- 所有设置仅保存在本地浏览器

### 系统要求

- Chrome 114 或更高版本
- 需要网络连接以使用翻译服务
