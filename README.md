# Google Translate 网站

一个功能完整的在线翻译工具，提供现代化的用户界面和强大的翻译功能。(GORDON)

## 🌟 主要功能

### 翻译功能
- **多语言支持**: 支持100+种语言的互译
- **实时翻译**: 输入文本时自动翻译（可关闭）
- **高质量翻译**: 使用Google Translate API提供准确翻译
- **语言检测**: 自动检测输入文本的语言

### 用户界面
- **左右布局**: 清晰的输入和输出区域
- **响应式设计**: 支持桌面和移动设备
- **深色/浅色主题**: 可自动跟随系统主题
- **现代化UI**: 使用Material Design风格

### 实用工具
- **语音朗读**: 支持文本朗读功能
- **一键复制**: 快速复制翻译结果
- **语言交换**: 一键交换源语言和目标语言
- **文本统计**: 显示字符数和词数
- **历史记录**: 自动保存翻译设置

## 🚀 快速开始

### 1. 打开网站
直接在浏览器中打开 `index.html` 文件即可使用。

### 2. 选择语言
- 点击左侧语言选择器选择源语言
- 点击右侧语言选择器选择目标语言
- 使用交换按钮快速切换语言

### 3. 开始翻译
- 在左侧输入框中输入要翻译的文本
- 翻译结果会自动显示在右侧
- 也可以点击"翻译"按钮手动触发翻译

## ⌨️ 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + Shift + T` | 翻译文本 |
| `Ctrl + Shift + S` | 交换语言 |
| `Ctrl + Shift + C` | 复制翻译结果 |
| `Ctrl + Shift + L` | 清除输入文本 |
| `Ctrl + Enter` | 在输入框中换行 |

## ⚙️ 设置选项

### 自动翻译
- 开启后，输入文本时会自动翻译
- 关闭后，需要手动点击翻译按钮

### 显示音标
- 开启后，会显示音标信息（如果可用）

### 主题设置
- **浅色主题**: 明亮的界面
- **深色主题**: 护眼的深色界面
- **自动主题**: 跟随系统主题设置

## 🔒 隐私说明

- 所有翻译请求直接发送到Google Translate
- 本地存储仅保存用户设置偏好
- 不收集或存储用户输入的文本内容
- 支持无痕浏览模式

## 🚧 注意事项

### API限制
- 使用Google Translate的免费API
- 可能有请求频率限制
- 建议在生产环境使用官方Google Cloud Translation API

### 浏览器要求
- 需要支持ES6+的现代浏览器
- 需要网络连接以访问翻译服务
- 语音功能需要浏览器支持Web Speech API
